# AI Schedule Recommendation – Frontend

This repository contains the **frontend UI** for the AI-based section/schedule recommendation system, built using **NiceGUI**.

## Overview
The application guides the user through three steps:
1. **Upload Data** – Students upload course, faculty, and timeslot files
2. **AI Analysis** – Backend processes data and analyzes constraints
3. **Results** – Optimized schedule options are displayed

## Tech Stack
- Python 3.9+
- NiceGUI

## How to Run Locally
```bash
pip install nicegui
python app.py
